How Green Coin Works

Detailed workflow for Users, Factories, NGOs, and Admin - each playing a crucial role in our ecosystem

User Journey

Register & Login
Create account and access dashboard
Submit Plant Proof
Upload tree photos with location data
NGO Verification
Wait for NGO approval of tree proof
Earn Credits
Receive 5 Green Credits per verified tree
Sell Credits & Get Money
Convert credits to real money through admin
Access: My Proof page, Leaderboard, Transaction history

Factory Process

Register & Login
Create factory account and access dashboard
Buy Credits Request
Submit request to purchase carbon credits
Admin Verification
Wait for admin approval of purchase request
Receive Credits
Get carbon credits for pollution offset
Pollution Data Report
Submit environmental impact reports
Access: Credit Balance Report, Purchase History

NGO Operations

Register & Login
Create NGO account with verification
Access Proof Verification
Review submitted tree planting proofs
Verify/Reject Proofs
Approve or reject tree planting evidence
Update User Credits
Automatically award credits for verified trees
Role: Independent verification authority

Admin Control

Middle Man Operations
Facilitate transactions between users & factories
Approve User Sell Requests
Process user credit sales and provide money
Approve Factory Buy Requests
Process factory purchases and send credits
Manage Users & Factories
Block/delete accounts, view all transactions
Full access: All transactions, credit transfers, user management
